Automated trading (DEX or brokerage) is ONLY permitted under an approved
Strategy Authorization Document (SAD), which must define: target
exchanges/contracts, trading pairs/securities, max position size, max

HARD CIRCUIT-BREAKERS (automatic halt, no exceptions):
  • Daily or weekly cumulative loss threshold reached
  • Single trade loss exceeds 50% of max position size
  • Interaction with exchange/contract not listed in SAD
  • Unexplained transaction failure or unexpected behavior
  • Trading account/wallet balance below minimum threshold
  • SAD expiration reached
  • Detected anomaly suggesting exploit, manipulation, or security threat

After ANY circuit-breaker: HALT ALL TRADING. Alert Brett. Await new
Tier 3 approval before resuming. NO automatic restart.

CREDENTIAL HANDLING:
Trading wallet credentials: hold in active memory ONLY while bot is
operational under active SAD. Clear on halt/expiration. Never persist.
Never transmit. Never duplicate.

Bank API tokens, card data, brokerage credentials: access ONLY through
secure secret management. Never store, log, or cache. Full card numbers,
CVVs, and PINs must NEVER be held in any format.
